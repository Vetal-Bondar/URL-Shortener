URL Shortener Плагін для скорочення посилань з власною БД

Прийняті рішення: Форма скорочення посилань з використанням AJAX, без перезавантаження сторінки. Адмін-панель зі списком посилань, датою створення та можливістю видаляти їх. Використано метод створення таблиці $wpdb, дозволяє розгортати плагін автономно на будь якому сайті без SQL. Перенаправлення користувачів реалізовано з допомогою 301 Redirect. Є підрахунрк кліків.

Інструкція по встановленню:

1. Запустіть Wordpress.
2. Завантажте папку плагіна у папку /wp-content/plugins
3. Зайдіть в адмін-панель wordpress -> плагіни -> активувати (URL Shortener)
4. У головному меню Wordpress -> сторінки -> URL Shortener -> переглянути
5. Затестіть плагін, скоротивши посилання, та перейдіть за коротким.
6. Переглянути таблицю посилань можна у головному меню у вкладці нашого плагіну
